# ruby-chess
A command-line chess game implemented in Ruby. Made during the CodeCore 2015 bootcamp.

## Move sequences for testing

### Check
c2 -> c3; d7 -> d5; d1 -> a4

### Checkmate
f2 -> f4; e7 -> e5; g2 -> g4; d8 -> h4

### Castling - Player 1
b1 -> a2; a7 -> a6; b2 -> b3; b7 -> b6; c1 -> b2; c7 -> c6; c2 -> c3; d7 -> d6; d1 -> c2
